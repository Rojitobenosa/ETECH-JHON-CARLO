Lesson Summary

Summary of Lessons for the Semester

1. Mail Merge

Learn how to use mail merge for automating personalized letters, emails, or documents by merging data sources into templates.

2. Text Wrap

Understand how to position and format text around images or objects within a document.

3. Slide/Image Presentation

Master the creation of slideshows and the inclusion of images to enhance visual communication.

4. Information Graphics or Infographics

Explore techniques for designing infographics that present data visually in an appealing and easily digestible way.

5. Image Manipulation

Practice editing and transforming images using software to improve or modify visuals.

6. Creation of Website

Develop skills to build a basic website, from designing layouts to adding content.

7. Advocacy for Social Change

Learn how to use ICT tools to create campaigns that promote societal awareness and action.

8. Petition Using Change.org

Gain experience in creating and managing online petitions to gather support for a cause.

9. How to Publish Your ICT Project

Understand the steps to make your ICT project publicly accessible through various publishing platforms.

10. Analyze Your ICT Project Using Facebook Page

Utilize Facebook analytics to evaluate the impact and engagement of your ICT project.

11. Creation of Feedback Using Google Forms

Learn how to create and distribute surveys to collect feedback and opinions.

12. Creation of Reflective Piece or Output

Practice writing reflective pieces or developing outputs summarizing learnings and experiences.
